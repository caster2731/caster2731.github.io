Kenmo FM

KenmoFMは、最新のAI技術とインターネット文化を融合させた、24時間365日稼働の自動ラジオ放送システムです。
5ちゃんねる（嫌儲板）の熱気あるスレッドから、古今の名作文学、そしてAIによる鋭いニュース解説まで、多彩なコンテンツを全自動でキュレーション・生成し、リスナーに届けます。

Concept

「インターネットの喧騒を、心地よいラジオへ」
KenmoFMは、テキストベースのインターネット上の議論や情報を、聴覚的なエンターテインメントへと昇華させます。
Google Cloud TTS、Voicevox、Bert-VITS2などの高度な音声合成技術を駆使し、単なる読み上げを超えた「ラジオ番組」としての体験を提供します。

Key Features

  • 全自動スレッド読み上げ & ラジオ化
    5ちゃんねる（嫌儲板）の勢いのあるスレッドをリアルタイムで取得。高性能フィルタリングと、時間帯に合わせたBGM/SE演出でプロのラジオのような聞き心地を実現。
    定時の時報やジングル、そして「嫌儲十訓」の定期放送も。
  • AIニュース解説・コメンタリー
    Gemini API / NotebookLMを活用し、最新ニュースをAIが独自の視点で解説。AI Duoモードによるキャラクター同時の掛け合いで、深みのある議論を展開します。
  • 多彩な専門コーナー
    深夜の「怪談コーナー」、情感豊かな「文学コーナー」、そしてAI DJが盛り上げる音楽放送「Musicthon」。
  • ポッドキャスト自動配信
    放送されたコンテンツの一部は自動的にパッケージ化され、Apple PodcastsやSpotifyへ配信されます。

Technical Highlights

  • Multi-TTS Engine: Google Cloud TTS (Chirp3, Neural2), Voicevox, Bert-VITS2 (実験的モデル)
  • Automated Mixing: FFmpegとPygameを駆使したリアルタイムかつ高品質な音声ミキシング
  • Expandability: 新しいコーナーを柔軟に追加できるプラグイン的アーキテクチャ

Vision

インターネットの「今」を切り取り、AIの力で再構築する。
KenmoFMは、作業用BGMとして、あるいは情報収集ツールとして、リスナーの生活に寄り添う新しいメディアの形を提案します。

Server Log Cockpit

Nginxのアクセスログ（標準 Combined 形式）を解析し、美しいダッシュボードで視覚化するWebアプリケーションです。
PV・UU・転送量・エラー率を一目で確認でき、サーバー管理者やWeb開発者の強力な分析ツールとなります。

Key Features

  • ダッシュボード
    PV, UU, 転送量, エラー率を一目で確認。チャートで可視化。
  • 詳細リスト
    Top Clients（DNS逆引き＆国旗表示）、Top Requests、User Agents、Referers
  • ボットフィルター
    クローラーを除外して純粋なユーザーアクセスのみを分析
  • ライブモニター
    リアルタイムで「tail -f」のように監視

Additional Features

  • 🇯🇵 日本語対応: 日英切り替え可能
  • 📂 ファイルブラウザ: GUIでログファイル選択
  • 💎 モダンUI: ダークモード＆Glassmorphism

Gravity Storage 🪐

PC内のフォルダ構成を「太陽系」に見立てて3D視覚化するアプリケーションです。
フォルダは恒星（太陽）、ファイルは惑星として表現され、その「重力（サイズ）」を感じながら探索できます。

Key Features

  • 惑星メタファー
    ファイルの種類ごとに異なる惑星として表示（木星=動画、地球=画像など）
  • サイズと重力
    ファイルサイズが大きいほど惑星も巨大に描画
  • 3D探索
    マウス操作で回転・ズーム、フォルダクリックでドリルダウン

技術スタック

  • Frontend: React (TypeScript), Vite
  • 3D Library: Three.js, React Three Fiber
  • Backend: Python (FastAPI)
